📖 Template Syntax

{{name}}              → HTML-escaped variable
{{&name}}             → Raw variable (no escaping)
{{.}}                 → Current item in #each loop
{{#each items}}...{{/each}}  → Iterate over a slice
{{#if flag}}...{{/if}}       → Conditional
{{#if flag}}...{{else}}...{{/if}} → If/else

All templates are compiled at comptime — zero runtime parsing overhead.
